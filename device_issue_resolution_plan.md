## 问题1：设备主页左上角出现异常灰色方形显示

### 1.1 问题分析

通过代码审查，发现问题根源：
- **位置**: `Live2DView.kt:57`
- **原因**: `Box` 组件设置了灰色背景色 `Color.LightGray`，且当宠物位置未正确初始化时，会显示在左上角 (0, 0) 位置
- **相关代码**:
  ```kotlin
  Box(
      modifier = modifier
          .size(120.dp)
          .background(Color.LightGray) // 问题代码
          .offset { IntOffset(position.x.roundToInt(), position.y.roundToInt()) }
  )
  ```

## 问题2：设备手势交互功能无响应

### 2.1 问题分析

通过代码审查，发现潜在问题：
- **位置**: `Live2DView.kt`
- **潜在原因**: 
  1. 两个独立的 `pointerInput` 修饰符可能导致手势冲突
  2. 手势检测范围可能被其他组件遮挡
  3. 事件消费逻辑可能有问题

| 拖拽 | 移动宠物位置 | 待验证 |
